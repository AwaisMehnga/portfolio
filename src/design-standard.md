# Portfolio Design Standard

## Font Families
- `font-sans`: Primary UI font
- `font-mono`: Skill tags / technical labels

## Color Tokens
- `bg-surface` / `text-ink`: Base surfaces and primary text
- `text-muted`: Secondary text
- `border-border`: Standard border
- `border-brand`, `text-brand`, `bg-brand-soft`: Brand interaction states

## Type Scale
- `text-display`: Hero name / major headlines
- `text-title`: Section titles
- `text-subtitle`: Supporting subheadings
- `text-chip`: Skill tags / labels
- `text-button`: Button labels

## Radius + Shadows
- `rounded-card`: Primary card blocks
- `rounded-chip`: Tags/chips
- `rounded-button`: Buttons
- `shadow-chip`: Default floating tag shadow

## Spacing Variables
- `--ds-space-container-x`: Horizontal page padding
- `--ds-space-section-y`: Vertical section spacing
- `--ds-space-hero-pad`: Hero center block spacing

## Rule
Use these tokens/classes first before adding one-off values so all sections stay visually consistent.
